# 한자 훈 (뜻) 음 (읽기) 부수 획수 급수 삭제
🔑 API 키 설정

Anthropic Claude API 키를 입력하세요. 키는 브라우저 localStorage에만 저장되며, 영어 어휘 앱과 공유됩니다 (vocabApiKey).

⚙️ Cloudflare Worker 업데이트

핸드폰 촬영 기능을 사용하려면 기존 vocab-proxy Worker에 아래 코드를 추가해야 합니다.

  1. Cloudflare 대시보드 → Workers & Pages → vocab-proxy
  2. Edit code 클릭 후 아래 코드를 기존 코드에 병합
  3. Save and Deploy
